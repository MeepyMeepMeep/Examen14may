Para alojar (o compartir) un repositorio en GitHub, siga los siguientes pasos:
Paso 1: Registrar una cuenta en GitHub
El primer paso para comenzarcon el alojamiento en GitHub es crear una cuenta personal. Visite la página de registro oficial para registrarse.

Paso 2: Crear un repositorio remoto en GitHub.
Después de registrarse, crear un home (un repositorio) en GitHub para el repositorio Git que quieres compartir.

Paso 3: Conectar el directorio Git del proyecto con el repositorio remoto.
Una vez que hayas creado un repositorio remoto para tu proyecto, tienes que vincular el directorio del proyecto.
